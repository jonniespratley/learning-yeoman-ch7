## Getting Started

Install the module with: `npm install jps-n`

```js
var jps-n = require('jps-n');
jps-n.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g jps-n
$ jps-n --help
$ jps-n --version
```


```sh
# creates a browser.js
$ grunt browserify
```
